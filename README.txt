# goat99 PWA (iPhone)

## Как обновить GitHub Pages
1) Открой репозиторий: https://github.com/sm3rch1337-lab/goat99
2) Замени файлы в корне репозитория:
   - index.html
   - manifest.webmanifest
   - sw.js
   - папку icons/
3) Подожди 1-2 минуты, обнови страницу.

## Как поставить на iPhone 14
1) Открой Safari → https://sm3rch1337-lab.github.io/goat99/
2) Кнопка "Поделиться" → "На экран «Домой»"
3) Открой приложение с рабочего стола (оно будет работать как обычное приложение).

## Где хранятся данные
- Данные сохраняются на этом iPhone (localStorage + IndexedDB).
- В настройках есть "Резервная копия":
  - Экспорт → скачивает backup.json
  - Импорт → восстановление из файла или текста
